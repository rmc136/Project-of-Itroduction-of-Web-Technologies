Pequena história do jogo

Minesweeper era um jogo incluído nos sistemas operativos Windows (desde o 3.1 até ao Windows 7), em que o objetivo era descobrir todas as minas existentes num campo de quadradinhos.

Como Jogar

O tabuleiro é dividido em células, com as minas distribuídas aleatoriamente. Para vencer, é necessário abrir todas as células que não contêm minas. Clicar numa célula que não tem uma mina revela um número. Este número é a quantidade de células vizinhas que contêm uma mina. Usando essas informações, pode-se determinar células que são seguras e células que contêm minas. As células suspeitas de serem minas podem ser marcadas com uma bandeira.
